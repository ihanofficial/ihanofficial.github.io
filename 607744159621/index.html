10 信息政策与信息法律
2025-05-17 06:34:26 # 考研复习 # 专业课 # 信息管理学基础 #信息管理

国家信息政策:
在一国范围内，国家或政府为解决信息管理和信息经济发展中出现的关系和涉及公共利益、权益、安全问题、保障信息活动协调发展而采取的有关信息产品及资源生产、流通、利用、分配以及促进和推动相关信息技术发展的一系列措施、规划、原则或指南。

国家信息政策的范围：

  1. 确定社会的信息需求
  2. 设计适当的方法以便使信息需求得到满足
  3. 促进信息资源的有效利用
  4. 信息专业人员的人力发展
  5. 确定必需的基础建设
  6. 确保持续的经费支持

国家信息政策分类：

  1. 为制定信息政策提供的规范性框架和指南
  2. 信息政策方面的具体研究报告和咨询报告
  3. 以国家信息政策为研究对象的一般性理论性研究，以及信息政策活动的大范围综述或述评
  4. 与信息技术发展有关的信息政策

信息政策的作用：

  1. 确定社会信息活动的发展方针，指明信息事业的大方向，指导信息事业的发展，为社会信息活动提供具有导向性和约束力的行动准则
  2. 调动或约束社会信息力量，促进信息资源的合理开发和有效利用
  3. 协调信息环境系统各部门与外部环境的相互关系，保证社会信息环境的健康发展
  4. 干预信息资源的建设和信息资源的利用过程
  5. 规范信息工作的标准化

信息政策的五要素

主体:

目标:是通过制定信息政策所实现的利益协调和分配关系。

问题:是指能够列入政策制定议程的客观社会问题，即信息政策应该解决的社会矛盾和问题。

政策内容：

政策形式：是政策内容的外在表现形式，是宣传、执行政策的有效工具和合法依据。

我国信息政策五要素的分析：

  1. 政策主体
  2. 政策目标：
  3. 政策问题：
    1. 经济问题
      1. 宏观层次
        1. 信息产业发展与投资：
        2. 信息基础设施建设
        3. 信息资源管理：
        4. 信息技术管理：
        5. 人为资源的投资
        6. 信息经济测度标准和信息化指标的制定问题
      2. 微观层次
        1. IT行业、网络产业的垄断问题。
        2. 出版业的高额利润问题
    2. 法律问题
      1. 解决信息共享和知识产权保护的矛盾问题
        1. 工业产权保护、专利权保护、技术转让
        2. 版权、著作权保护
        3. 商标权、域名保护
      2. 解决信息自由流通和信息安全的矛盾问题
        1. 信息公开与信息保护问题
        2. 国际信息合作与国家信息主权的关系
        3. 计算机病毒、黑客程序防范与计算机网络安全问题
        4. 电子商务的电子签名和数字认证问题
  4. 政策内容
    1. 经济层面
      1. 宏观产业发展经济问题
      2. 微观市场经济问题
    2. 法律层面
      1. 知识产权保护
      2. 信息自由流通与信息安全
  5. 政策形式：

政策实施机构与政策制定机构隔离的优势：

  1. 可以减少利益集团的影响，从而减少不确定性，并且可以制定出公平的政策；
  2. 具有独立性的政策制定机构将更加趋于专业化；
  3. 可以增加决策的透明性。
